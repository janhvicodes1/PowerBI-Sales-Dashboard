# Sales Analytics Dashboard - Power BI

## Overview
Built a Power BI sales analytics dashboard to analyze sales performance, customer activity, and product trends.

The dashboard provides insights into revenue patterns, top-performing products, and category-wise sales distribution.

## Tools Used
- Power BI
- DAX
- Data Cleaning
- Data Visualization
- Data Modeling

## Dashboard Features

- KPI tracking:
  - Total Sales
  - Total Orders
  - Total Customers

- Time-based sales analysis:
  - Yearly and quarterly sales trends

- Product analysis:
  - Top-selling products
  - Product performance comparison

- Category analysis:
  - Sales distribution across product categories

- Interactive filtering:
  - Date range
  - Country
  - Product category

## Project Workflow

1. Imported sales dataset into Power BI
2. Cleaned and transformed data
3. Created relationships between tables
4. Built calculated measures using DAX
5. Designed dashboard visuals to analyze business performance
